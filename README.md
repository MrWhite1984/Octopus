<h1>Octopus — KVP (Key-Value Pair) СУБД с графическим интерфейсом</h1>
Octopus — локальная база данных, ориентированная на хранение данных в формате ключ → значение. Система состоит из двух компонентов:
<ul>
  <li><b>OctoServ</b> — серверная часть, управляющая хранением и доступом к данным;</li>
  <image src="git_img/UI1.png">
  <li><b>Octopus UI</b> — графический клиент для взаимодействия с базами.</li>
  <image src="git_img/UI2.png">
</ul>
<em>Статус: разработка приостановлена. Функционал ограничен, архитектура может изменяться.</em>
<h2>Текущие возможности</h2>
<ul>
  <li>Создание и выбор между несколькими именованными базами данных через выпадающий список.</li>
  <image src="git_img/UI3.png">
  <li>Просмотр содержимого выбранной базы в виде таблицы (колонки: Ключ, Значение).</li>
  <li>Режим Консоль.</li>
  <image src="git_img/UI4.png">
  <li>Сервер (OctoServ.exe) запускается отдельно.</li>
</ul>
<h2>Технологии</h2>
Язык программирования: <b>C#</b><br>
GUI: <b>Windows Forms</b><br>
Архитектура: <b>локальный клиент-сервер</b><br>
Хранение: <b>файловая система</b>
